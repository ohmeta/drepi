# Dereplicate genomes or genes

## Instalation

```sh
conda install drepi
```

## Run
### help

```sh

```

### init

```sh
drepi init -d . -s genes.tsv
# or
drepi init -d . -s genomes.tsv
```

### run

#### dereplicate genes
```
drepi drep_genes all
```

#### deprelicate genomes

```
drepi drep_genomes all
```
